Helpsni 0.4 – mobil, zakázky, chat, platby a hodnocení

Funkce:
- mobile-first rozhraní pro iPhone a Android
- spodní mobilní navigace a bezpečné okraje
- úprava, smazání a archiv zakázek
- chat s fotografiemi a stavem Odesláno/Přečteno
- testovací Apple Pay, Google Pay a platební karta
- provize Helpsni 10 %
- oboustranné hodnocení po archivaci zakázky
- service worker připravený pro skutečné web push notifikace

Důležité:
- současná verze ukládá data do prohlížeče (localStorage)
- platby jsou pouze simulace, skutečné peníze se nestrhávají
- vzdálené push notifikace mezi zařízeními vyžadují backend a VAPID klíče
- ověření pracovníka nikdy nevyžaduje občanku ani selfie s doklady
